# 📱 Smart Shop Management System (C# & SQL Server)

## 📝 Description
An advanced desktop application designed for retail tech shops (Smart Shops). This project focuses on **dynamic data filtering** and a modern user experience for managing electronics and gadgets.

## 🚀 Key Features
*   **🔍 Dynamic Filtering:** Search and filter products by category, price range, brand, or availability in real-time.
*   **📦 Inventory Control:** Automated stock alerts and product management.
*   **🛒 Sales Module:** Quick checkout process with automated receipt generation.
*   **📊 Advanced Reports:** Visual overview of sales performance using SQL aggregation.
*   **🔐 User Roles:** Secure login for Employees and Admin dashboards.

## 🛠 Tech Stack
*   **Language:** C# (.NET Framework)
*   **Database:** Microsoft SQL Server (Optimized stored procedures/queries)
*   **UI Framework:** Windows Forms (WinForms)
